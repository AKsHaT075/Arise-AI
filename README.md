# Arise AI

Arise AI is a Bharat-first AI AR educational mobile application that
helps fresh students and new hires understand debugging through visual execution,
AR-based guidance, and AI-powered explanations.

## Problem
Code execution and debugging feel invisible and abstract for beginners, making
learning slow and confusing.

## Solution
Arise AI combines booklet-based learning, on-device OCR, AR visualization, and
Amazon Bedrock powered AI explanations to act as a pocket mentor for learners,
even in low-connectivity environments.
